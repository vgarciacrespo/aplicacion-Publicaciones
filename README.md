# Aplicación de Publicaciones

Esta es una aplicación móvil y de escritorio desarrollada con **.NET MAUI** para la gestión de publicaciones científicas y autores.

## Características

- **Gestión de Publicaciones**: Permite añadir, listar, modificar y eliminar publicaciones.
- **Gestión de Autores**: Permite registrar nuevos autores en el sistema.
- **Búsqueda Avanzada**: Posibilidad de buscar publicaciones por autor o por tipo (Libro, Capítulo, Artículo, Congreso).
- **Control de Estados**: Seguimiento del estado de las publicaciones (Revisión, Aceptado, Rechazado).

## Requisitos

- [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0) o superior.
- [Visual Studio 2022](https://visualstudio.microsoft.com/vs/) con la carga de trabajo de .NET MAUI instalada.

## Instalación y Ejecución

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/vgarciacrespo/Aplicacion-Publicaciones.git
   ```
2. Abrir la solución `app/PracticaGrupalEnero.sln` en Visual Studio.
3. Restaurar los paquetes NuGet y ejecutar en el emulador o dispositivo deseado.
